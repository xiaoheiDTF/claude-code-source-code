# 12 - 安全与权限 (Security & Permissions)

## 对应源码
- `src/constants/`（权限限制常量）
- `src/utils/permissions.ts`
- 各工具目录内的安全检查逻辑

## 关键安全机制

| 机制 | 说明 |
|------|------|
| 工具权限检查 | 每个工具执行前的权限验证 |
| 路径验证/消毒 | 文件操作前的路径安全检查 |
| 命令沙盒 | BashTool 的安全命令执行 |
| API 限额 | apiLimits.ts / toolLimits.ts |
| 用户确认 | 敏感操作前的用户确认机制 |

## 架构角色
贯穿整个应用的**安全防线**——从工具执行、文件访问到 API 调用，每个层级都有权限控制。
